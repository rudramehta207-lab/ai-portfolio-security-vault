# 🛡️ Enterprise AI Portfolio Factory & Cryptographic Vault

A high-security, multi-tier Python application that leverages Artificial Intelligence to generate stunning professional developer portfolios, while simultaneously acting as a secure cryptographic vault. The system allows users to embed highly confidential data into image assets using military-grade encryption and steganography.

## 🚀 Key Security & Architecture Features

* **AES-256 Symmetric Encryption:** Secures text payloads using advanced cipher frameworks (CFB mode) with unique salt and Initialization Vectors (IV) before pixel injection.
* **LSB Steganography Engine:** Injection of encrypted binary data streams straight into the Least Significant Bits of image pixels with localized delimiter markers.
* **Interactive UI Workspace Preview:** Built with Streamlit to offer real-time rendering of generated web components through an integrated preview window.
* **Automated QR Verification Matrix:** Dynamically constructs and embeds data-integrity QR validation codes directly into the downloadable responsive HTML outputs.
